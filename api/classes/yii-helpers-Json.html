Json extends BaseJson
in package
  • Default

Json.php : 17

Json is a helper class providing JSON data encoding and decoding.

It enhances the PHP built-in functions json_encode() and json_decode() by supporting encoding JavaScript expressions and throwing exceptions when decoding fails.

encode()

BaseJson.php : 58

Encodes the given value into a JSON string.

public static encode(mixed $value[, int $options = 320 ]) : string

The method enhances json_encode() by supporting JavaScript expressions. In particular, the method will not encode a JavaScript expression that is represented in terms of a [[JsExpression]] object.

Note that data encoded as JSON must be UTF-8 encoded according to the JSON specification. You must ensure strings passed to this method have proper encoding before passing them.

Parameters
$value : mixed

the data to be encoded.

$options : int = 320

the encoding options. For more details please refer to https://secure.php.net/manual/en/function.json-encode.php. Default is JSON_UNESCAPED_SLASHES | JSON_UNESCAPED_UNICODE.
