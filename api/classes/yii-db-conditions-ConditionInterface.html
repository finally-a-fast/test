ConditionInterface extends ExpressionInterface
in
  • Default

ConditionInterface.php : 20

Interface ConditionInterface should be implemented by classes that represent a condition in DBAL of framework.

The database abstraction layer of Yii framework supports objects that implement this interface and will use [[ExpressionBuilderInterface]] to build them.

The default implementation is a class [[Expression]].
