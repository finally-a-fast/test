Message extends BaseMessage
in package
  • Default

Message.php : 36

Message implements a message class based on SwiftMailer.

By default, [[send()]] will use the "mailer" application component to send the current message. The "mailer" application component should be a mailer instance implementing [[MailerInterface]].
